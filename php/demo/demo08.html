static 关键字

static 静态关键字

静态变量和全局变量本质上是没有区别的
静态变量只能在类里面进行使用 体现了面向对象的封装性
静态变量 是放在类中或者 函数里面的

存在方式

存在于静态常量去 不再堆区存地址了
public static $num = 0; //定义并初始化了 静态变量 加做 $num 空间存在的是地址 值在静态常量去 所以能够引用

访问

self :: $num += 1; // 去使用静态变量
// Child::$num 类中这样也可以   离开只能这种
$p1 ->num; 报错 不允许这样访问静态变量 但是java可以

细节

/当我们操作静态变量的时候 我们考虑使用静态方法 比如统计 所有学生交的学费
// 静态对象属于 所有对象实例的 不区分对象 在我们变成中 往往使用静态方法操作变量
// 静态方法不能操作非静态属性
