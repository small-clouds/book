函数和变量

变量的的定义

变量：变量是存储信息的容器
$a= 10;

变量的规则

变量以 $ 符号开头，其后是变量的名称
变量名称必须以字母或下划线开头
变量名称不能以数字开头
变量名称只能包含字母数字字符和下划线（A-z、0-9 以及 _）
变量名称对大小写敏感（$y 与 $Y 是两个不同的变量）
注PHP 变量名称对大小写敏感！

变量的作用域

php中变量是弱类型语言 通过赋值 才能决定变量的数据类型
php变量有三种作用域(local(局部) global(全局) static(静态))
global(全局) 函数之外声明的变量拥有Global作用域
关键词用于访问函数内的全局变量。 要做到这一点，请在（函数内部）变量前面使用 global 关键词 global $x,$y;(函数内使用)
local(局部) 函数之内声明的变量拥有LOCAL作用域
注：???您可以在不同的函数中创建名称相同的局部变量，因为局部变量只能被在其中创建它的函数识别。
static 关键词 通常，当函数完成/执行后，会删除所有变量。不过，有时我需要不删除某个局部变量。实现这一点需要更进一步的工作。 要完成这一点，请在您首次声明变量时使用 static 关键词
注： 该变量仍然是函数的局部变量。

常量

常量类似变量，但是常量一旦被定义就无法更改或撤销定义。
注释：与变量不同，常量贯穿整个脚本是自动全局的
如需设置常量，请使用 define() 函数 - 它使用三个参数：define("GREETING", "Welcome to W3School.com.cn!",true);
首个参数定义常量的名称
第二个参数定义常量的值
可选的第三个参数规定常量名是否对大小写不敏感。默认是 false。
也可以这样定义： const MIN_NUM = 0;

函数

function funName(参数列表){
函数体(必须有);
return(可有可无)；
}
细节：
参数列表：只是形式参数 简称形参
不可以同时引入两个function 否则会报重名错误
程序看到函数 就会开辟一个新栈 各栈间的变量是相互独立的
return 能够返回结果 没有renturn 也返回 只是没有返回值
// 参数可以多个
// 可以php支持的任意类型
// 函数名不区分大小写 function abc 和function ABC 同一个函数
函数内局部变量
function abc($n) { if ($n > 2) { abc( --$n); } echo $n; } abc(4); //开辟新栈 真有意思 2 2 3 对值 --了 所以 第二次变成了2 所以没有4
错误级别 notice最轻的 warn 警告 error 致命的
函数的深入使用
function abc2($b, $a = 2) 默认值 不传为2 传值为穿进去的值

函数传参数 地址与值

在php中如果传递的是数组 在默认情况下传递的是值 重新拷贝了一份数组
&$a 在变量前加上&符 传递的将是地址 就可能改变原来的值
