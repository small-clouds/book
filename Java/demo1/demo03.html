js实现全选和反选

onclick="selectNo()"  // onclick="selectAll()"
function selectAll() {
    var arr = document.getElementsByName("hobby");
    for (var i = 0; i < arr.length; i++) {
        arr[i].checked = true; // 这里没有双引号
    }
}
function selectNo() {
    var arr = document.getElementsByName("hobby");
    for (var i = 0; i < arr.length; i++) {
        if (arr[i].checked == true) {
            arr[i].checked = false;
        } else {
            arr[i].checked = true;
        }
    }
}

节点

注：不同的元素节点会有自己本身的属性 例如width color之类的等等不胜枚举

节点的分类

文档节点(document)
元素节点(element)
文本节点(text)

节点的属性

firstChild: childNodes 列表中第一个节点。
lastChild: childNodes
parentNode: 返回父节点
previousSibling: 返回当前节点之前的节点。（直接父节点）
nextSibling: 返回当前节点的下一个节点。(直接后兄弟节点)
nodeName: 返回节点的HTML标签名称，使用英文的大写字母，例如：p,
fontnodeType: 返回节点类型，1为标签，2为属性，3为文字节点
specified: 返回一个布尔值，返回HTML标签是否设置了指定的属性值

节点的方法

getAttribute(name) 返回名为 name 的属性值
removeAttribute(name) 删除名为 name 的属性
setAttribute(name, value) 创建一个名为 name 的属性并将其值设为 value
getAttributeNode(name) 返回名为 name 的属性节点
removeAttributeNode(node) 删除与指定节点匹配的属性节点
createElement(elementName) 使用给定的名称创建一个元素
createTextNode(text) 使用提供的文本创建一个新的文本节点
createAttribute(attributeName) 用提供的名称创建一个新属性
insertBefore(newChild, referenceNode) 将 newChild 节点插入到 referenceNode 之前。
replaceChild(newChild, oldChild) 用 newChild 节点替换 oldChild 节点。
removeChild(oldChild) 从运行该方法的节点中删除 oldChild 节点。
appendChild(newChild) 将 newChild 添加到运行该函数的节点之中。
newChild 被添加到目标节点孩子列表中的末端。
hasChildNodes() 在调用该方法的节点有孩子时则返回 true，否则返回 false。
hasAttributes() 在调用该方法的节点有属性时则返回 true，否则返回 false。
