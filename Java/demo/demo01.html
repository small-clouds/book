安装

安装JDk
安装完成后会询问是否安装jre 取消即可 jdk是包含jre
傻瓜式安装，一路确定 一路下一步
注： 路径不能有中文，不能有特殊字符
验证：在jdk的bin目录下 输入命令javac 出现提示说明安装成功

java环境的配置

作用：在任何目录下都能使用javac命令，所以需要配置环境变量
环境变量位置：计算机属性->高级系统设置->高级(选项卡)->环境变量
变量JAVA_HOME 值(jdk的安装路径): C:\Program Files\Java\jdk1.7.0_09;
变量PATH 值：%JAVA_HOME%\bin; 动态获取bin
变量CLASSPATH 值： ;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar;
