数据库的数据类型

字符串型 varchar char varchar(20) 变化的字符串 最多存20 省空间 效率低 / char(12) 定长字符串 费空间效率高
大数据类型 ：以前 blob 字节型大数据 存二进制文件 / text 字符型大数据 存文档 不能存图片 文字
数值型 ： int / tinyint / smallint / int/ bigint / double / double / float
日期型 ：date 年月日 / time 时分秒 / datetime 年月日时分秒 / timestamp年月日时分秒的时间戳

DML 增删改

增：insert into 表名(id,name, male,age) values (); / insert into 表名 values（）;
成倍增长： insert into users (name,brith,salary,remark) select name,brith,salary,remark from users;
改：update user set name=‘王五’ where age =9 and male =0; 没有条件将更改所有
删：delele from 表名 where 条件
截断表: truncate table user 清空表 但是数据从头开始的

DQL

简单查表：select name, age from user;
起别名： select u.name u.age, u.male from user (as) u； // 表其别名
区间查找一：select name ， chinese from in　（20,19）；
区间查找二：select name chinese from __ between __ and__ ;
模糊查询(like)：select * from user where name like '-明%'; ////// % 任意位置
空数据： select name ， chinese from user is (not) null
排序： select * from order by (desc); (order by 放到语句的最后)
