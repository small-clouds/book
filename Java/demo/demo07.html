数组---学习集合类之前唯一的能够放置大规模数据的地方

数组类型

是用来存储相同数据类型的数据 ； 因此数组的类型就是存储的数据的数据类型

数组定义形式

第一种
类型[] 标识符 = new 类型[长度]
int[] arr = new int[5];
第二种
String[] arr ={"a","b","c","d"};
第三种
double[] arr =new double[]{1.1,1.2,1.3,1.4}

数组的功能 存取

存：长度从1开始 下标是数组的编号 下标从0开始 arr[0] =10;
取：根据下标取值 arr[0] arr[1]] 正序 0 -5

增强for循环

foreach 和for的区别
注：for-each 必须知道数组的类型。(万物皆对象)for则不需要
    for（String s:arr）{
        System,out.println(s);
    }

二维数组

就是数组里面还是数组
int[][] arr={{1,2,3,4},{5,6,7,8}}
取值无非就是两层循环
