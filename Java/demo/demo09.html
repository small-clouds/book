面向对象二

方法/函数

public void 方法名(参数){}
main主方法 必须用static修饰
分类：有参/无参，有返回值/无返回值 四类

关于返回值

有返回值的方法 必须有return 有且只能有一个
返回值的类型 必须与定义的类型一致
返回值的个数 有且只能有一个

构造函数/构造方法

每个类 都有一个默认的无参构造方法 没有返回值类型 所以也没有返回值
作用： 用于创建对象
分类:有参构造 / 无参构造
在类中如果不显示的写出任何构造器 那么类默认提供一个无参构造方法
当类中编写了有参构造方法是 不默认提供无参构造方法 所有 最好提供一个无参构造方法

重载

方法名相同 参数列表（个数和类型）不同的 方法之间互称 重载 与返回值和修饰符没有关系

重写

子类与父类方法 相同 那么子类的方法的优先级高首先引用 本类的方法 不再引用父类的方法了 这就是方法的重写
super 子类中直接父类的引用
