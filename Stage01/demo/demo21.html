设计模式

单例模式

A对象加载到内存当中 所以判断内存当中 只能有一个

懒汉式

public class A{
    private static A a;
    private A(){
        super();
    }
    public static A getInstance(){
        if(a ==null){
            a = new A();
        }
        return a;
    }
}

饿汉式

public class B{
    private static B b = new b();
    private B(){
        super();
    }
    public static B getInstance(){
        return b;
    }
}
